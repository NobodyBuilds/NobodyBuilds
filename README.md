## 💫 About Me

I build real-time physics simulations from the ground up. No engines. No abstractions hiding the math.

Obsessed with how nature actually works — and making silicon replicate it correctly. Governing equations first, performance second, always. CUDA/C++ is the tool. Correctness is the non-negotiable.
